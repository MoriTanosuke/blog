--- layout: post title: USB unter Java ---

Am Wochenende habe ich mich kurz mit dem aktuellen Stand zum USB-Zugriff unter Java umgesehen. Und ich muss sagen: Es sieht gar nicht gut aus.

libusb4j ist bereits on hold, der Fork libusb4j auf Project Kenai sieht auch tot aus. Ausserdem kann man nach dem Auschecken so gut wie gar nichts damit anfangen...

Ich habe mich noch ein wenig auf Github umgesehen und auch da findet man diverse USB-Schnittstellen, die aber allesamt eher nach interner Entwicklung aussehen. Jedenfalls hat mich bei keiner ein Juhu-das-will-ein-ausprobieren-Gefül überkommen.

Da ich aber unbedingt auf ein USB-Gerät zugreifen möchte, für das es keine geeignete Software für mich - aber eine Umsetzung in Python gibt, muss ich wohl oder übel meine eigene Bibliothek anfangen. Basierend auf JSR80 und JNA, hoffentlich anständig gekapselt für Linux und später für Windows.

Aber bis ich mich entschlossen habe, werde ich noch weiter suchen und vielleicht mal auf StackOverflow nachfragen.
